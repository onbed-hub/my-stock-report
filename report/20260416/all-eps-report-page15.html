🚀 利潤擴張ΔEPS > ΔRev > 0規模經濟優化：獲利動能領先營收。代表產品具定價權或固定成本大幅攤提。
📈 穩健同步ΔRev ≈ ΔEPS > 0健康營運：獲利隨業務等比例增加，經營效能維持在穩定水位。
⚠️ 獲利稀釋ΔRev > ΔEPS > 0效率警訊：生意變大但利潤成長跟不上，受成本上升或削價競爭影響。
📉 獲利侵蝕ΔRev > 0 且 ΔEPS < 0邊際貢獻崩壞：營收增、獲利跌。經營陷入「白忙一場」的結構性惡化。
🧐 疑業外收益ΔRev < 0 且 ΔEPS > 0獲利虛胖：本業衰退但EPS上升。需嚴防資產處分或匯兌收益一次性干擾。
🔄 轉虧為盈Prev EPS < 0 < EPS基期反轉：從虧損轉為獲利，需觀察獲利是否具備續航力。
