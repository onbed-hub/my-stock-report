📊 獲利品質與估值準則

❶ 估值參考指標

指標名稱計算邏輯判讀方式
本益比 (PE)現價 / 近四季 EPS 總和反映回本年數。15 倍為合理中軸，低於 12 倍相對便宜。
15X 推估價近四季 EPS 總和 × 15量化合理價基準。若目前股價低於此值，代表具備估值優勢。

💡 關鍵計算：利潤空間 (Profit Expansion)

算法： 利潤空間 = EPS 年增率 (YoY) - 營收年增率 (YoY)

解釋： 此指標用來衡量「獲利成長是否跑贏營收成長」。
• 大於 0% (正值)： 代表「獲利品質提升」。可能是毛利率拉高、費用率降低或具備規模經濟優勢。
• 小於 0% (負值)： 代表「獲利遭到侵蝕」。生意雖然變大，但被成本、費用或低價競爭吃掉利潤。

❷ 經營策略深度解析

診斷標籤數學模型經營策略深度解析
🚀 利潤擴張ΔEPS > ΔRev > 0規模經濟優化：獲利動能領先營收。代表產品具定價權或固定成本大幅攤提。
📈 穩健同步ΔRev ≈ ΔEPS > 0健康營運：獲利隨業務等比例增加，經營效能維持在穩定水位。
⚠️ 獲利稀釋ΔRev > ΔEPS > 0效率警訊：生意變大但利潤成長跟不上，受成本上升或削價競爭影響。
📉 獲利侵蝕ΔRev > 0 且 ΔEPS < 0邊際貢獻崩壞：營收增、獲利跌。經營陷入「白忙一場」的結構性惡化。
🧐 疑業外收益ΔRev < 0 且 ΔEPS > 0獲利虛胖：本業衰退但EPS上升。需嚴防資產處分或匯兌收益一次性干擾。
🔄 轉虧為盈Prev EPS < 0 < EPS基期反轉：從虧損轉為獲利，需觀察獲利是否具備續航力。
